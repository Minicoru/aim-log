# aim-log

## Description

This is a NPM module to generate logs in a local file y the root of the app where is used.

## usage

```javascript
const aimlog = require("aim-log");

aimlog("test");
aimlog({ test: "test" });
aimlog(["test", "test"]);
aimlog(1);
aimlog(1.1);
aimlog(true);
aimlog(false);
aimlog(null);
aimlog(undefined);
aimlog(NaN);
aimlog(Infinity);
aimlog(-Infinity);
aimlog(Symbol("test"));
aimlog(() => {});
aimlog(async () => {});
aimlog(function () {});
aimlog(class {});
aimlog(new Date());
aimlog(new Error());
aimlog(new Map());
aimlog(new Set());
aimlog(new WeakMap());
aimlog(new WeakSet());
aimlog(new Promise(() => {}));
aimlog(new Proxy({}, {}));
aimlog(new ArrayBuffer(2));
aimlog(new Int8Array(2));
aimlog(new Uint8Array(2));
```

## Installation

As simple as any module of NPM, you just need to execute in your terminal the next command.

```bash
npm install aim-log
```
